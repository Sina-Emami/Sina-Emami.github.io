Our Project

This model is trained to segment those parts which are infected by Covid-19 in CT-Scan.

  • Trained with COVID-19 CT scans Kaggle data set.
  • For the loss function, the combination ofweighted dice loss and surface losswas used and model trained with U-Net.
  • Used Histogram Equalizationto make infection parts more visible and used augmentation to create more data for model.
  • Sensitivity in test data set reached 0.934817 and AUC reached 0.933751.
  • Tested with CT-MD Dicom data set and gets the perfect result as output.

  • Link: GitHub
  • Date: Oct 2020
  • Service: TensorFlow, Keras, Pandas, OpenCV, Numpy
